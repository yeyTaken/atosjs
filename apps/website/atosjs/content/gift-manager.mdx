# Configurando o GiftManager

<Box type="info" title="Duas formas de configurar o GiftManager" isDropDown>
  Existem duas abordagens principais para configurar o GiftManager no seu projeto:
  1. **Modo Padrão:** Instancia o GiftManager diretamente onde for necessário.
  2. **Modo Recomendado:** Centraliza a configuração em um arquivo específico.

    ## Modo Padrão

    Essa abordagem instancia o **GiftManager** diretamente onde ele for necessário.

    ```js filename="index.js"
    const { GiftManager } = require('atosjs');

    const gm = new GiftManager();
    ```

    ### Estrutura de Arquivos
    <FileTree>
      <FileTree.Folder name="my-project" defaultOpen>

  <Tabs.Tab>
    ## Modo Recomendado

    A abordagem recomendada centraliza a configuração do **GiftManager**, facilitando manutenção e escalabilidade.

    **Passo 1: Criando a configuração**
    ```js filename="atos.config.js"
    const { GiftManager } = require('atosjs');

    const gift = new GiftManager({ dblocal: "./gifts.sqlite" });

    module.exports = { gift };
    ```

    **Passo 2: Importando a configuração**
    ```js filename="index.js"
    const { gift } = require('./atos.config');
    ```

    ### Estrutura de Arquivos
    <FileTree>
      <FileTree.Folder name="my-project" defaultOpen>
        <FileTree.File name="atos.config.js" />
        <FileTree.File name="index.js" />
      </FileTree.Folder>
    </FileTree>

## Parâmetros do GiftManager

Os parâmetros podem ser configurados para personalizar o comportamento do **GiftManager**:

```js
const gm = new GiftManager({
  dblocal: "./gifts.sqlite", // Caminho para o banco de dados local (SQLite ou JSON)
//  mongodb: { 
//    connect: "mongodb://localhost:27017", URL do banco MongoDB
//    dbName: "meuBanco" Nome do banco de dados no MongoDB
//  }
});
```

- **`dblocal`** *(opcional)*: Define um banco de dados local (`.sqlite`, `.json`, `.db`, `.yaml`, `etc...`).
- **`mongodb`** *(opcional)*: Configuração para armazenar os dados no MongoDB.
  - **`connect`**: URL de conexão com o MongoDB.
  - **`dbName`**: Nome do banco de dados que será utilizado.
