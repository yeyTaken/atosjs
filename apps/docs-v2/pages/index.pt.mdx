## Sobre
AtosJS é um pacote poderoso e fácil de usar, desenvolvido com [Node.js](https://nodejs.org/) e compatível com ESLint para garantir o uso correto de importações. Ele oferece sistemas úteis para tarefas diárias, otimizando seu fluxo de trabalho. Visite o [site](https://atos.js.org/).

## Classes
- `GiftManager` é uma ferramenta para criar, gerenciar e resgatar códigos de recompensa integrados a APIs como Discord Bot, facilitando o engajamento da comunidade.
- `TimeFormat` é uma classe para agendar e repetir tarefas em intervalos ou horários específicos.
