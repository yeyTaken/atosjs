## Sobre
AtosJS é um pacote poderoso e fácil de usar, desenvolvido com [Node.js](https://nodejs.org/) e compatível com ESLint para garantir o uso correto de importações. Ele oferece sistemas úteis para tarefas diárias, otimizando seu fluxo de trabalho. Visite o [site](/).

## Classes
- [`GiftManager`](/docs/gift-manager/) é uma ferramenta para criar, gerenciar e resgatar códigos de recompensa integrados a APIs como Discord Bot, facilitando o engajamento da comunidade.
- [`TimeFormat`](/docs/time-format/) | [`ms`](/time-format/) é uma classe para agendar e repetir tarefas em intervalos ou horários específicos.
- [`AtosDB`](/docs/atos-db/) é uma database baseada em SQLite.
- [`PokeDex`](/docs/pokedex/) é uma classe que coleta as informações dos pokemons.
