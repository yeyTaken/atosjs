## About

AtosJS is a powerful and easy-to-use package developed with [Node.js](https://nodejs.org/) and compatible with ESLint to ensure the correct use of imports. It offers useful systems for daily tasks, optimizing your workflow. Visit the [website](/).

## Classes

* [`GiftManager`](/docs/gift-manager/) is a tool to create, manage, and redeem reward codes integrated with APIs like Discord Bot, enhancing community engagement.
* [`TimeFormat`](/docs/time-format/) is a class for scheduling and repeating tasks at specific intervals or times.
* [`@atosjs/database`](/docs/atos-db/) is an SQLite-based database.
* [`@atosjs/azura`](/docs/azura/) is a high-performance framework for building APIs.
* [`@atosjs/pokemon`](/docs/pokedex/) is a class that gathers information about Pok√©mon.
