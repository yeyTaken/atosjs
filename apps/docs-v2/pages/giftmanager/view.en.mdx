## Method `view`

<Callout type="info" emoji="ℹ️">
The `view` method allows you to view detailed information about a generated gift, including its validity and associated values.
</Callout>

The `view` method from the `GiftManager` class is used to view detailed information about a generated gift. It returns an object containing information about the validity and data of the gift.

---

### Method Signature
```js showLineNumbers filename="JavaScript" copy
async view(giftId: string): Promise<Object>
```

---

### Parameters
- **`giftId`** (string): The unique identifier of the gift to be viewed. This ID is generated by the `generate()` method.

<Callout type="warning" emoji="⚠️">
Make sure the `giftId` passed is valid and has been generated by the same instance of `GiftManager`.
</Callout>

---

### Return
A `Promise` that resolves to an object containing:
- **`valid`** (boolean): Indicates whether the gift is still valid.
- **`type`** (string): The type of the gift.
- **`value`** (string): The value associated with the gift.

<Callout type="info" emoji="ℹ️">
The returned data allows you to verify the gift's validity before attempting to redeem it.
</Callout>

---

### Usage Example

The example below demonstrates how to use the `view` method to obtain detailed information about a gift:

```js showLineNumbers filename="JavaScript" copy
async function gift() {
    const gm = new GiftManager({
        fileName: 'atosjs-test',
        fileType: 2 // YAML format
    });

    const giftId = await gm.generate({
        type: 'test',
        value: 'test giftId in string',
        maxRedeem: 3,
        edit: {
            code: 'ATOSJS_TEST'
        }
    });

    console.log({ 
        giftID: giftId,
        giftView: await gm.view(giftId), // Displays detailed information about the gift
    });
};

gift();
```

<Callout type="default" emoji="💡">
If you only need a specific property, you can directly access it using dot notation.
</Callout>

```js showLineNumbers filename="JavaScript" copy
async function gift() {
    console.log({ 
        giftViewType: await gm.view(giftId).type, // Displays the gift type.
        giftViewValid: await gm.view(giftId).valid, // Displays `true` for valid or `false` for invalid.
    });
};

gift();
```
