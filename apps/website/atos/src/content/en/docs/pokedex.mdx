## Basic Usage

```javascript filename="index.js" showLineNumbers copy
const { PokeDex } = require("@atosjs/pokemon");

const pokedex = new PokeDex();

(async () => {
  // Fetching Pok√©mon by name
  const pikachu = await pokedex.fetch({ name: "pikachu" });
  
  // All Pok√©mon information
  console.log("Pikachu:", pikachu);
  
  // Displaying the sprite
  console.log("Sprite:", pikachu.sprite);
})();
````

<Callout type=".">
The example above demonstrates how to instantiate the `PokeDex` class and fetch the Pok√©mon "pikachu". You can also fetch using `id` or even arrays of values.
</Callout>

## Multiple Queries

```javascript filename="multi-fetch.js" showLineNumbers copy
const { PokeDex } = require("@atosjs/pokemon");

const pokedex = new PokeDex();

(async () => {
  // Fetching multiple Pok√©mon using id
  const pokemonList = await pokedex.fetch({ id: [1, 4, 7] });
  console.log("Pok√©mon list:", pokemonList);
  
  // Or using names
  const anotherList = await pokedex.fetch({ name: ["bulbasaur", "charmander", "squirtle"] });
  console.log("Another list:", anotherList);
})();
```

<Callout type=".">
You can also pass arrays with multiple values for `id` or `name`, and you will get an array with the structured data of each Pok√©mon.
</Callout>

## üß© Return Structure (`PokeData`)

The structured response returned by the `PokeDex` class has the following format:

| Field             | Type                    | Description                                       |
| ----------------- | ----------------------- | ------------------------------------------------- |
| `name`            | `string`                | Pok√©mon's name.                                   |
| `id`              | `number`                | Pok√©mon's national ID.                            |
| `type`            | `string[]`              | Pok√©mon's types.                                  |
| `abilities`       | `string[]`              | Pok√©mon's abilities.                              |
| `sprite`          | `string`                | URL of the front image (sprite).                  |
| `height`          | `number`                | Height in decimeters.                             |
| `weight`          | `number`                | Weight in hectograms.                             |
| `base_stats`      | `Record<string,number>` | Base stats (HP, attack, defense, etc.).           |
| `evolution_chain` | `{ url: string }`       | URL of the evolution chain.                       |
| `description`     | `string`                | Descriptive text taken from the API (in English). |
